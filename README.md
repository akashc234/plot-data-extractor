### ☀️ Run Flask Server

* Create a virtualenv and activate:
  ```bash
  $ virtualenv venv && source venv/bin/activate
  ```
* Install python dependencies
  ```bash
  $ pip install -r requirements.txt
  ```
* Run
  ```bash
  $ python app.py
  ```
* The application will be accessible at `http://localhost:5000`
